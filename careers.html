Full Stack Java Developer Intern
Location: Remote
Salary: 6,000-8,000 BDT/month (Based on skills)
Type: Internship
Duration: 4 Months
Office Hours: 10:00 AM - 6:00 PM
Job Description

We are looking for a motivated Full Stack Java Developer Intern to join our development team. You will work on building and maintaining our business management platform, gaining hands-on experience with modern Java technologies and contributing to real-world applications used by businesses globally.

This is an excellent opportunity for a recent graduate or student to kickstart their career in software development while working on impactful projects in a fast-growing startup environment.

Note: Since this is an internship position, we don't expect you to be a professional developer. We're looking for candidates who have basic programming knowledge, at least one personal project to showcase, and most importantly, a hunger to learn new technologies and grow as a developer.

Tech Stack
Java 24 Spring Boot Spring Data JPA PostgreSQL JTE HTMX Bootstrap Git GitHub Tomcat
Requirements
  • Bachelor's degree in Computer Science or related field (or current student)
  • Basic knowledge of Java programming and OOP concepts
  • Familiarity with Spring Boot framework (preferred)
  • Understanding of SQL and database concepts
  • Basic knowledge of HTML, CSS, and JavaScript
  • Experience with Git version control
  • Strong problem-solving skills and eagerness to learn
  • Good communication skills in English
Key Responsibilities
  • Develop and maintain web applications using Java and Spring Boot
  • Work with PostgreSQL database design and optimization
  • Implement responsive frontend interfaces using Bootstrap and HTMX
  • Collaborate with the development team using Git and GitHub
  • Write clean, maintainable, and well-documented code
  • Participate in code reviews and team meetings
  • Learn and adapt to new technologies as needed
What We Offer
  • Competitive internship stipend (6,000-8,000 BDT/month based on skills)
  • 4-month internship program with structured learning
  • 100% remote work flexibility
  • Mentorship from experienced developers
  • Opportunity to work on real-world projects
  • Potential for full-time conversion after internship
  • Learning and development opportunities
  • Flexible working hours within office hours
